**Dashboard 4: Churn Signals (INST-06)**
1. Go to PostHog -> Dashboards -> New dashboard
2. Name: "Churn Signals"
3. Add a HogQL insight (SQL mode):
```sql
SELECT
    person_id,
    min(timestamp) AS credit_limit_at
FROM events
WHERE event = 'credit limit reached'
  AND timestamp >= now() - interval 90 day
GROUP BY person_id
HAVING NOT EXISTS (
    SELECT 1 FROM events e2
    WHERE e2.event = 'credit purchased'
      AND e2.person_id = person_id
      AND e2.timestamp > credit_limit_at
      AND e2.timestamp < credit_limit_at + interval 14 day
)
```
   - Alternative approach: Use a Funnel with exclusion — Step 1: `credit limit reached`, exclusion: `credit purchased` within 14 days. Either approach works.
4. Save insight to dashboard
